# birthday-calendar

Birthday Calendar is a simple program that reads birthday data from an .xlsx file and produces representative data in .xlsx format.

The data used in the source .xlsx file are the House Block, Resident Name, and National Identification Number (NIK) columns. This program will automatically read the NIK data format and extract birthday date data from it.

The source .xlsx file is sensitive data and will never be shared publicly. If you need to customize the columns used as the source, you can do so directly in the code.
